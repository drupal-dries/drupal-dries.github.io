An event website launched!

March 11, 2024 Hikaru Maruyama

We created an event website. We were not initially planning to create one, but since we were going to go to the trouble. It is a multilingual website that can be used in Japanese and English, so we hope it will be a good opportunity for people in other countries to find out about the event.

The technical stack of the site includes. Thanks to the Field Group and Display Suite modules, I was able to build a layout with almost no code.

  • CMS: Drupal
  • Base theme: Barrio Bootstrap 5 Theme
  • Static site generator: Tome
  • Hosting: Github Pages

We are glad we were able to do this in time before the event. We will be updating the site with event-related updates in the future.
